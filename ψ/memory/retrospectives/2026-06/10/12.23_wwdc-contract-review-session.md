## Lessons Learned

1. **Two-source conflict** → เปิดไฟล์ใหม่ อย่าแก้ของเก่า: เมื่อมี 2 แหล่งข้อมูลขัดแย้งกัน การสร้างไฟล์ใหม่พร้อม fact-check table ดีกว่าการแก้ไข in-place ซึ่งอาจทำให้ประวัติหายหรือเกิดความสับสน

2. **PDF legal review**: แม้ font encoding จะ fragment text แต่ข้อมูลสำคัญ (วันที่, เงิน, ชื่อ, เลขบัญชี, ลายเซ็น) มักอยู่ใน readable section — focus ที่นั่นก่อนเสมอ

3. **ให้ tag ที่ใช้ได้จริง**: Tags ควรตอบคำถาม "ใช้ที่ไหน" ก่อน — social media tag ≠ file naming tag ≠ search tag
